Phar-Hosting Optimizer est un outil d'ingénierie système conçu pour maximiser les performances de Windows en purgeant les caches profonds et en optimisant la pile réseau.

## 🛠️ Fonctionnalités Élite
- **Purge Système** : Nettoyage des dossiers Temp (User & Win), Prefetch et Logs.
- **Network Reset** : Flush DNS, Register DNS et réinitialisation Winsock.
- **Force Mode** : Fermeture automatique des processus bloquants (Chrome, Edge, Firefox).
- **Auto-Privilège** : Gestion native de l'élévation de privilèges Administrateur.
- **UI Moderne** : Interface fluide basée sur CustomTkinter.
